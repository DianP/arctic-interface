# Changelog - January 5, 2026

## TUI Improvements

### Prompt Input Height

- Reduced maximum prompt input height to keep dialogs readable

### Dialog Prompt Listing

- Dialog prompt picker now shows multiple prompts instead of one

### Dialog Selection Copy

- Disabled auto-copy when selecting text inside dialogs

## Authentication

### Anthropic OAuth Flow

- Fixed Anthropic OAuth callback parsing and token refresh handling

## Configuration

### Provider Session Title Options

- Added provider config options to disable session or message titles

## Maintenance

### Cleanup

- Removed unused packages and assets

---

**Summary**: This release tightens the TUI prompt input height, improves prompt dialog listings, and removes auto-copy on dialog selections. It also fixes Anthropic OAuth handling, adds provider options for session/message titles, and includes repository cleanup.
